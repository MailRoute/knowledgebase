## MX Records: 1&1

  1. Log in to your account at https://admin.1and1.com.  
  2. If it's not already selected, click the **Administration** tab.  
  3. Under **Basic Settings**, click **Domains**.  
  4. Select the check box next to the domain that you will be using with MailRoute.  
  5. From the **DNS** drop-down menu, select **Edit DNS Settings**.  
  6. Under **Advanced DNS Settings**, select the **Other mail server** radio button.  
  7. Delete any existing MX records.   
  8. Enter the following information: 
    * MX 1
    * MailServer: mail.mailroute.net
    * Priority: 10 
  9. Click **OK**.
