## MX Record Configuration: webhost4life

  1. Log in to your account at webhost4life.
  2. Go to **DNS**.
  3. Select the domain name that you'd like to use with MailRoute.
  4. Click **Edit Zone**.
  5. Insert the following MX record with the associated priority values: 
    * MailServer: **mail.mailroute.net.** (Don't forget the trailing dot "."!)
    * Priority: **10**
