Mac OS X Server 10.4 or later can be locked down to only accept email from the MailRoute servers by following these steps:

1.  In **Server Admin**, select **Mail** in the **Computer & Services** pane
2.  Click **Settings**
3.  Select the **Relay** tab
4.  Check **Accept SMTP relays only from these**:
5.  Edit the list of hosts:
    * Click the Add (+) button to add a host to the list: 199.89.0.0/21

For more information about configuring  Mac OS X server, please see http://manuals.info.apple.com/en/Mail_Service_v10.4.pdf
